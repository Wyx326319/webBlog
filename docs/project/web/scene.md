## 1.如何判断设备类型(手机端，移动端，ipad),并判断浏览器的类型

### 第一种：使用 User-Agent 字符串
不同的浏览器和设备都会有不同的特征，而这些特征可以通过navigator.userAgent这个属性获取，这个属性包含了浏览器的版本，操作系统等信息，通过解析这个信息，我们可以
识别出大多数的浏览器和设备类型
