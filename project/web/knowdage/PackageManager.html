包管理器 ​

推荐一个Node.js版本控制管理工具 nvm ​

1.一定要卸载nodejs！！！,否则会导致nvm和node版本都不能使用或版本对不上！！！ ​

2.安装nvm ​

nvm安装地址 傻瓜式安装，一直点next就好！

3.安装node.js ​

打开命令行窗口,输入下列指令(此处以17.9.1版本为例)下载node.js

js
nvm install 17.9.1

使用node.js

js
nvm use 17.9.1

查询node版本

node -v

到此nvm就安装好了，可以随意的切换node版本！！ ​
